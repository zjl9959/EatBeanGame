# 游戏运行环境

## 安装以下 MOD

+ FlipFlop logic gates
+ Automation Expanded
+ Automation Plus
+ Blueprints fixed 【可选-方便批量操作】

## 打开 debug 模式

参考链接：https://www.gamersky.com/handbook/201705/906855.shtml

## 打开沙盒模式

参考链接：https://www.bilibili.com/read/cv27695653/

## 加载存档

+ 创建新游戏
+ 加载本地存档
+ 选择 `CPU_release.sav` 文件

# 编译器使用

## 安装 python 依赖

```shell
python -m pip install pyqt5
```
## 运行编译器

```shell
python compiler.py
```
